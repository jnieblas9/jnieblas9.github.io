QGIS en LINUX

¿Qué es QGIS?

QGIS es un sistema de información geográfica (SIG) de código abierto que permite el análisis de datos espaciales y la creación de mapas. Este manual te guiará a través de los pasos necesarios para instalar QGIS en una distribución de Linux basada en Ubuntu.

2. Prerrequisitos

Asegúrate de tener permisos de administrador en tu sistema y que los paquetes de software estén actualizados. Ejecuta lo siguiente en tu terminal:

sudo apt update && sudo apt upgrade

3. Instalación de QGIS

3.1. Añadir la Llave del Repositorio

Primero, agrega la llave de firma del repositorio de QGIS con el siguiente comando:

wget -qO - https://qgis.org/downloads/qgis-archive-keyring.gpg | gpg --dearmor | sudo tee /usr/share/keyrings/qgis-archive-keyring.gpg > /dev/null

3.2. Agregar el Repositorio de QGIS

Después, añade el repositorio correspondiente a tu versión de Ubuntu. Si usas Ubuntu 22.04 (Jammy Jellyfish), usa:

sudo add-apt-repository "deb [signed-by=/usr/share/keyrings/qgis-archive-keyring.gpg] https://qgis.org/ubuntu $(lsb_release -c -s) main"

Si tu versión no es compatible, puedes intentar usar el repositorio de focal (Ubuntu 20.04):

sudo add-apt-repository "deb [signed-by=/usr/share/keyrings/qgis-archive-keyring.gpg] https://qgis.org/ubuntu focal main"

3.3. Actualizar el Índice de Paquetes

Actualiza la lista de paquetes disponibles en los repositorios:

sudo apt update

3.4. Instalar QGIS y Plugins

Instala QGIS junto con el plugin de GRASS ejecutando:

sudo apt install qgis qgis-plugin-grass

4. Verificación de la Instalación

Una vez completada la instalación, puedes verificar si QGIS está disponible ejecutando:

qgis

Si QGIS se inicia correctamente, la instalación ha sido exitosa.

5. Solución de Problemas

  • Error de repositorio: Verifica que has añadido la línea correcta al archivo /etc/apt/sources.list.d/qgis.sources.
  • Errores de llave: Asegúrate de que la llave del repositorio fue añadida correctamente.
  • Paquetes faltantes: Ejecuta sudo apt-get install -f para resolver dependencias.

6. Conclusión

Con estos pasos, habrás instalado QGIS en tu sistema Linux y podrás comenzar a utilizar esta potente herramienta para análisis geoespacial. Si tienes más dudas, visita la documentación oficial de QGIS.
